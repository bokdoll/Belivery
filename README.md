# Belivery
__WIC 해커톤 powered by AWS : 안드로이드__

:eight_spoked_asterisk: **beli**eve(믿음직한) + deli**very**(배달) : 빌리버리

:point_right: 빌리버리는 1인 가구에 초점을 맞춰 그들이 기존에 배달원으로부터 느끼던 불안함과 불편함을 한번에 잡은 새로운 개념의 음식 배달 서비스입니다.
